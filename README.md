# Personal-Ticket-System
The Personal Ticket System is a lightweight web application designed to help me manage tasks and requests more effectively. Instead of keeping tasks in my head, every task is recorded as a ticket that can be created, updated, and closed.

This project is built as both:

A personal productivity tool
A software engineering learning project focusing on planning, documentation, and clean implementation
